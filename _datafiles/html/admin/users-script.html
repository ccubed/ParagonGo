User Script

A single global script that runs for every player. See the User Scripts documentation for event hooks, and the Scripting Functions reference for available APIs.

Available event handlers:

  • onLogin(user, room) — player entered the world.
  • onLogout(user, room) — player left the world.
  • onLevel(user, room, eventDetails) — player gained a level.
  • onCommand(cmd, rest, user, room) — runs before mob/room scripts; return true to halt the command.
  • onDying(user, room) — player downed/bleeding out; return true to suppress default handling.
  • onDie(user, room) — player fully died; return true to abort the death.
